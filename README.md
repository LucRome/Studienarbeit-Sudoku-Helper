# Setup
- Go to `src` folder and run:
```
python -m venv .venv
```
- After creating the virtual environment enter it:
    - Windows: `.venv\Scripts\activate.bat`
    - Unix: `source .venv/bin/activate`
- Install the dependencies:
    - `pip install -r requirements.txt`
